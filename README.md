# 🛍️ Myntra Clone (HTML & CSS)

## 📌 Project Overview

This project is a **Myntra-inspired e-commerce website clone** built using **HTML and CSS**. It replicates the basic layout and design of the Myntra homepage, focusing on styling, responsiveness, and user interface structure.

The goal of this project is to practice front-end development skills and understand how modern e-commerce websites are designed.

---

## 🚀 Features

* 🏠 Homepage with banner and promotional sections
* 🧭 Navigation bar with categories (Men, Women, Kids, etc.)
* 🛒 Product display grid
* 🔍 Search bar UI (non-functional)
* 📱 Responsive design (for different screen sizes)
* 🎨 Clean and modern UI inspired by Myntra

---

## 🛠️ Technologies Used

* **HTML5** – for structuring the web pages
* **CSS3** – for styling and layout
* Flexbox & Grid – for responsive design

---

## 📂 Project Structure

```
myntra-clone/
│── index.html

```

---

## 💡 How to Run the Project

1. Download or clone this repository
2. Open the project folder
3. Double-click on `index.html`
4. View the website in your browser
